## 8. 복합 생산성 지수 분석

이슈 처리 효율성과 코드 생산성을 결합한 복합 지수를 계산합니다.


### 8.0 복합 지수 산출 공식

각 복합 지수는 구성 지표를 **0–1 Min-Max 정규화** 후 단순 평균으로 계산합니다.

```
정규화(m) =
  (value - min) / (max - min)          # [+] 지표 (클수록 좋음)
  1 - (value - min) / (max - min)      # [-] 지표 (작을수록 좋음)

복합 지수 = mean( 정규화(m₁), 정규화(m₂), ... )
```

각 지수의 구성 지표와 방향은 다음과 같습니다.

| 복합 지수 | 구성 지표 | 방향 | 정규화 방식 |
| --- | --- | --- | --- |
| **이슈처리효율** | 이슈 처리율(Outflow/Inflow) | [+] | 그대로 사용 |
|  | 인당 이슈Outflow(해결) | [+] | 그대로 사용 |
| **코드생산성** | 인당 코드 변경량 | [+] | 그대로 사용 |
|  | 인당 개발산출물Outflow(해결) | [+] | 그대로 사용 |
| **리뷰활동** | 인당 Review수 | [+] | 그대로 사용 |
|  | 인당 Reject수 | [+] | 그대로 사용 |

> 지수 범위: 0.0(최저) ~ 1.0(최고). 주차 내 일별 값의 평균으로 표시됩니다.
