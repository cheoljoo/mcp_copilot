## 📐 계산식 설명

### Hexa Index 계산
```
1. 각 지표별 정규화 점수 (0~100 척도):
   - 높을수록 좋은 지표 (sign=+1):
     정규화점수 = (value - 기간최솟값) / (기간최댓값 - 기간최솟값) × 100
   - 낮을수록 좋은 지표 (sign=-1):
     정규화점수 = 100 - (value - 기간최솟값) / (기간최댓값 - 기간최솟값) × 100

2. Hexa Index = 유효 지표 정규화점수의 평균값
   - 범위: 0 ~ 100점 (높을수록 좋음)
   ⚠️  Hexa Index의 절댓값(예: 47.74점)은 분석 기간 내 min/max에 상대적인 값으로,
      기간이 바뀌면 기준점이 달라지므로 절댓값 자체는 비교 의미가 없음.
      의미 있는 것은 '추세(slope의 방향과 p-value)'이며,
      Hexa Index가 전반적으로 오르는지 내리는지로 방향을 파악하는 용도임.

3. 트렌드 분석 (선형회귀):
   - scipy.stats.linregress(날짜_일수, 정규화점수) 으로 slope, p-value 산출
   - slope > 0 → 개선 추세 / slope < 0 → 악화 추세
   - p-value < 0.1 → 통계적으로 유의미한 추세

4. 원본값 비교 (초기 7일 평균 → 최근 7일 평균):
   - 정규화 이전 실제 지표값 기준으로 얼마나 변했는지 확인
   - sign=+1: 값이 커지면 개선 / sign=-1: 값이 작아지면 개선
```
