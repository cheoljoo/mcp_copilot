## 분석 방법론

### Hexa Index 계산 방법

```
1. 각 지표별 퍼센타일 순위 계산:
   percentile_rank = rank(value) / total_rows × 100  (0~100)

2. 방향성 보정 (sign 적용):
   adjusted = percentile_rank         (sign = +1, 높을수록 좋음)
   adjusted = 100 - percentile_rank   (sign = -1, 낮을수록 좋음)

3. Hexa Index = mean(adjusted) across all valid metrics
   → 값이 높을수록 전반적으로 더 좋은 상태를 의미함
```

### 유의미성 검정 방법

```
- 기준선: 최근 주를 제외한 모든 주의 값
- 검정 방법: One-sample t-test (최근 주 값 vs 기준선 분포)
- 유의수준: p-value < 0.1 → 유의미한 변화
- z-score = (최근값 - 기준선 평균) / 기준선 표준편차
- 개선/악화 판단: (최근값 - 기준선 평균) × sign > 0 → 개선, < 0 → 악화
```
