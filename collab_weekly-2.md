## 📘 분석 용어 해설

### slope (기울기)란?
> 시계열 데이터에 선형회귀(OLS)를 적합시켰을 때 얻는 **주당 변화율**입니다.

| 항목 | 설명 |
| ---- | ---- |
| **slope > 0** | 지표값이 주마다 평균적으로 증가하는 추세 |
| **slope < 0** | 지표값이 주마다 평균적으로 감소하는 추세 |
| **slope의 절댓값** | 클수록 변화 속도가 빠름 |
| **[+] 지표에서 slope > 0** | 개선 중 (예: 이슈 처리율 증가) |
| **[-] 지표에서 slope < 0** | 개선 중 (예: 잔여이슈 감소) |
| **[+] 지표에서 slope < 0** | 악화 중 (예: 이슈 처리율 감소) |
| **[-] 지표에서 slope > 0** | 악화 중 (예: 잔여이슈 증가) |

> ⚠️ slope는 **크기의 단위가 지표마다 다르므로** 같은 지표끼리만 비교하세요.

---

### p-value (유의확률)란?
> 회귀 기울기(slope)가 **우연히 발생할 확률**입니다. 낮을수록 트렌드가 신뢰할 수 있음을 의미합니다.

| 구간 | 판정 | 의미 |
| ---- | ---- | ---- |
| **p < 0.05** | ✅ **통계적으로 유의** | 관측된 트렌드가 우연이 아닐 가능성이 95% 이상 |
| **p ≥ 0.05** | ⬜ **비유의** | 트렌드 방향이 우연일 수 있으므로 해석에 주의 |

> 본 보고서에서 **유의 기준: p < 0.05** 를 사용합니다.

### UNIT 시계열 트렌드 점수 기준

| 판정 | 점수 | 기호 |
| ---- | ---- | ---- |
| 악화 + 통계적 유의(p<0.05) | **−5** | 🔴 악화(유의) |
| 악화 + 비유의(p≥0.05) | **−3** | 🟠 악화(비유의) |
| 개선 + 비유의(p≥0.05) | **+1** | 🟡 개선(비유의) |
| 개선 + 통계적 유의(p<0.05) | **+2** | 🟢 개선(유의) |
| 데이터 부족 | **0** | ⬜ |
